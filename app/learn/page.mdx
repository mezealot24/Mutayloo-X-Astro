# ยินดีต้อนรับสู่ Learning Session

หน้านี้เขียนด้วย MDX (Markdown + JSX) ค่ะ ใช้สำหรับสรุปความรู้และแบบฝึกหัดสั้นๆ ให้คิวอ่านง่าย ไม่ overload ค่ะ

## โครงสร้างโฟลเดอร์พื้นฐาน

- `app/layout.tsx` — layout หลักของทั้งแอป
- `app/page.tsx` — หน้าแรก `/`
- `app/dashboard/*` — หน้าสำหรับแดชบอร์ด
- `app/learn/*` — พื้นที่สำหรับบทเรียน (MDX)

## ภารกิจถัดไป

1. เปิดหน้าแดชบอร์ด `/dashboard`
2. กลับมาที่ `/learn` เพื่อทำแบบฝึกหัดต่อ

> เคล็ดลับ: เราจะใช้ MCP Next Devtools เพื่อดึงความรู้ Next.js 16 ล่าสุดระหว่างเรียนค่ะ
